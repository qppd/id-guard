# Delete one passcode

**`https://euapi.ttlock.com/v3/keyboardPwd/delete`**

Delete a random passcode or custom passcode，there are two methods to do it：

##### （1）Delete by APP nearby the lock via bluetooth：

You can develop your own APP based on our [APP SDK](/document/doc?urlName=appSdkV3/androidSdkDemoV3/downloadEn.html)，call [Delete passcode](/document/doc?urlName=appSdkV3/androidSdkDemoV3/androidExample/passcodeEn.html) method to delete the passcode from the lock，then request this API ( with `deleteType=1`) to delete it from the cloud.

##### （2）Delete by cloud API via [gateway](/document/doc?urlName=userGuide/gatewayEn.html) or [WiFi lock](/document/doc?urlName=userGuide/wifiLockEn.html)：

If the lock is [WiFi lock](/document/doc?urlName=userGuide/wifiLockEn.html) or it's connected to a [gateway](/document/doc?urlName=userGuide/gatewayEn.html), you can delete the passcode remotely，just request this API ( with `deleteType=2`) .

### 1 Request example

xxxxxxxxxx

POST, ContentType:application/x-www-form-urlencoded

xxxxxxxxxx

curl \--location \-g \--request POST 'https://euapi.ttlock.com/v3/keyboardPwd/delete' \\

\--data\-urlencode 'clientId=fd2ff35ee3xxxxxx' \\

\--data\-urlencode 'accessToken=780f7671b6e6bxxxxxxx' \\

\--data\-urlencode 'lockId=24451' \\

\--data\-urlencode 'keyboardPwdId=10237' \\

\--data\-urlencode 'deleteType=2' \\

\--data\-urlencode 'date=1626945087000'
