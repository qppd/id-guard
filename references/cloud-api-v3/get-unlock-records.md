# Get unlock records

**`https://euapi.ttlock.com/v3/lockRecord/list`**

Query the records of a lock uploaded to the cloud server, can be filtered by operate time.

Notice: The cloud server only save the records in the last 6 months, earlier records can't be queried.

### 1 Request example

xxxxxxxxxx

curl \--location \--request GET 'https://euapi.ttlock.com/v3/lockRecord/list?clientId=4773aa036f7f49c68d876bb4be85c80c&accessToken=dfd5489d0cee31f0bdfaf59d0d42d71f&lockId=163377&startDate=1526674054000&endDate=1626674054000&pageNo=1&pageSize=20&date=1626674054000'
