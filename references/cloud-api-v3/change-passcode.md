# Change passcode

**`https://euapi.ttlock.com/v3/keyboardPwd/change`**

There are two methods to change the passcodeÔºö

##### Ôºà1ÔºâChange by APP nearby the lock via bluetoothÔºö

You can develop your own APP based on our [APP SDK](/document/doc?urlName=appSdkV3/androidSdkDemoV3/downloadEn.html)Ôºåcall [Modify Keyboard Password](/document/doc?urlName=appSdkV3/androidSdkDemoV3/androidExample/passcodeEn.html) method to change the passcode in the lockÔºåthen request this API ( with `changeType=1`) to sync it to cloud.

##### Ôºà2ÔºâChange by cloud API via [gateway](/document/doc?urlName=userGuide/gatewayEn.html) or [WiFi lock](/document/doc?urlName=userGuide/wifiLockEn.html)Ôºö

If the lock is [WiFi lock](/document/doc?urlName=userGuide/wifiLockEn.html) or it's connected to a [gateway](/document/doc?urlName=userGuide/gatewayEn.html), you can change the passcode remotelyÔºåjust request this API ( with `changeType=2`) .

With this API, you can change the name, valid period or passcode itself.

When you want to change passcode name, only `keyboardPwdName` parameter is needed, changeType is noot needed;

When changing valid period `startDate` and `endDate` parameters are requiredÔºõ

When changing passcode`newKeyboardPwd` parameter is required.

### 1 Request example

xxxxxxxxxx

POST, ContentType:application/x-www-form-urlencoded

xxxxxxxxxx

curl \--location \-g \--request POST 'https://euapi.ttlock.com/v3/keyboardPwd/change' \\

\--data\-urlencode 'clientId=fd2ff35ee3xxxxxx' \\

\--data\-urlencode 'accessToken=780f7671b6e6bxxxxxxx' \\

\--data\-urlencode 'lockId=24451' \\

\--data\-urlencode 'keyboardPwdId=123456' \\

\--data\-urlencode 'keyboardPwdName=Passcode for Tony' \\

\--data\-urlencode 'newKeyboardPwd=689768' \\

\--data\-urlencode 'startDate=1626945087000' \\

\--data\-urlencode 'endDate=1846945087000' \\

\--data\-urlencode 'changeType=2' \\

\--data\-urlencode 'date=1626945087000'
