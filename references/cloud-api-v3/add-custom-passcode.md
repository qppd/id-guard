# Add custom passcode

**`https://euapi.ttlock.com/v3/keyboardPwd/add`**

Add a 4-9 digits custom passcode to the lock，there are two methods to do it：

##### （1）Add by APP nearby the lock via bluetooth：

You can develop your own APP based on our [APP SDK](/document/doc?urlName=appSdkV3/androidSdkDemoV3/downloadEn.html)，call [Add passccode](/document/doc?urlName=appSdkV3/androidSdkDemoV3/androidExample/passcodeEn.html) method to wirte a passcode into the lock，then request this API ( with `addType=1`) to add it to the cloud to facilitate follow-up management.

##### （2）Add by cloud API via [gateway](/document/doc?urlName=userGuide/gatewayEn.html) or [WiFi lock](/document/doc?urlName=userGuide/wifiLockEn.html)：

If the lock is [WiFi lock](/document/doc?urlName=userGuide/wifiLockEn.html) or it's connected to a [gateway](/document/doc?urlName=userGuide/gatewayEn.html), you can add custom passcode remotely，just request cloud API [Add custom passcode](/document/doc?urlName=cloud/passcode/addEn.html) ( with `addType=2`) , the gateway will write the passcode into the lock.

If the WiFi lock is in power-saving mode, it can perform asynchronous data transmission (addType=4). For more information on asynchronous data transmission, please refer to:[WiFi lock](https://euopen.ttlock.com/document/doc?urlName=userGuide%2FwifiLockEn.html)

Due to the limited storage space of the lock, only 250 passcodes can be added at most. If the number exceeds the limit, an error code of - 3009 will be returned, indicating that the space is full. You need to delete others before adding them.
