# Query lock settings

**`https://euapi.ttlock.com/v3/lock/querySetting`**

Query lock settings via [gateway](/document/doc?urlName=userGuide/gatewayEn.html) or [WiFi lock](/document/doc?urlName=userGuide/wifiLockEn.html), including：2-Privacy lock、3-Tamper alert、4-Reset button、7-Open direction.

-   For other lock settings, such as sound on/off and volume, please use the "[Get Lock Details](https://euopen.ttlock.com/document/doc?urlName=cloud%2Flock%2FdetailEn.html)" API.
    
-   For WiFi lock settings, such as power-saving mode on/off, please use the "[Get WiFi Lock Information](https://euopen.ttlock.com/document/doc?urlName=cloud%2FwifiLock%2FdetailEn.html)" API.
    
-   For [door sensor](https://euopen.ttlock.com/document/doc?urlName=cloud%2FdoorSensor%2FqueryEn.html) and [passage mode](https://euopen.ttlock.com/document/doc?urlName=cloud%2Flock%2FgetPassageModeConfigurationEn.html), please use their respective APIs.
