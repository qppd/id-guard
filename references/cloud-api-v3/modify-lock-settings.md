# Modify lock settings

**`https://euapi.ttlock.com/v3/lock/updateSetting`**

Modify lock settings, including: 2-Privacy lock、3-Temper alert、4-Reset button、6-Lock sound、7-Open direction、8-Sound volume、10-Wifi lock's power saving（Note: Wifi lock's power saving mode can only be set via bluetooth, it can't be set it via gateway）.

-   If you have [modified the lock setting via APP SDK](/document/doc?urlName=appSdkV3/androidSdkDemoV3/lockApisEn.html), you can request this API to upload the setting value to cloud server；If the lock is [WiFi lock](/document/doc?urlName=userGuide/wifiLockEn.html) or it's connected to [gateway](/document/doc?urlName=userGuide/gatewayEn.html), you can request this API to modify lock settings remotely.
    
-   You can query modified lock settings by [Get lock details](/document/doc?urlName=cloud/lock/detailEn.html).
