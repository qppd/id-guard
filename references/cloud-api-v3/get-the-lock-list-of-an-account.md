# Get the lock list of an account

**`https://euapi.ttlock.com/v3/lock/list`**

List locks added by you，no matter it is added by TTLock APP or added by APP SDK and initialized by cloud API: [Lock init](/document/doc?urlName=cloud/lock/initializeEn.html)

-   If your locks were added by TTLock APP，request this API with the [access token get](/document/doc?urlName=cloud/oauth2/getAccessTokenEn.html) with the username and password of the user account you login TTLock APP.
    

-   If your locks were initialized by cloud API:[Lock init](/document/doc?urlName=cloud/lock/initializeEn.html), request with the same access token when init the lock.
    

**This API will only return locks which you are top administrator**. Ekeys others shared to you can be queried by cloud API: [ekey list](/document/doc?urlName=cloud/ekey/listEn.html).

If integrated with TTHotel, this API can also retrieve lists of power savers or lift controls. Simply add the parameter 'type': type = 1 for lock list, type = 2 for lift control list, and type = 3 for power saver list.
