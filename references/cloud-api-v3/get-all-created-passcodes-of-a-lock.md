# Get all created passcodes of a lock

**`https://euapi.ttlock.com/v3/lock/listKeyboardPwd`**

List all passcodes of a lock, including random passcodes and custom passcodes.

### 1 Request example

xxxxxxxxxx

curl \--location \-g \--request GET 'https://euapi.ttlock.com/v3/lock/listKeyboardPwd?clientId=fd2ff35ee3d8424c8665c07b7b9a7f45&accessToken=780f7671b6e6be3010d9787f19207aca&lockId=1719&pageNo=1&pageSize=20&date=1625025703000'
