# Get ekeys of a lock

**`https://euapi.ttlock.com/v3/lock/listKey`**

List all the common ekeys of a lock, including authorized ekeys.

This API will not return the top administrator's ekey.

Do you want to share an ekey to somebody? refer to cloud API: [Send ekey](/document/doc?urlName=cloud/ekey/sendEn.html).

### 1 Request example

xxxxxxxxxx

curl \--location \-g \--request GET 'https://euapi.ttlock.com/v3/lock/listKey?clientId=fd2ff35ee3d8424c8665c07b7b9a7f45&accessToken=780f7671b6e6be3010d9787f19207aca&lockId=1719&pageNo=1&pageSize=20&date=1625025703000'
