# Get lock battery

**`https://euapi.ttlock.com/v3/lock/queryElectricQuantity`**

Get the lock battery of a lock remotely via [gateway](/document/doc?urlName=userGuide/gatewayEn.html) or [WiFi lock](/document/doc?urlName=userGuide/wifiLockEn.html).

### 1 Request example

xxxxxxxxxx

curl \--location \-g \--request GET 'https://euapi.ttlock.com/v3/lock/queryElectricQuantity?clientId=fd2ff35ee3d8424c8665c07b7b9a7f45&accessToken=f6ee29dda68d962243fb91b047d319d3&lockId=163377&date=1548141056000'

### 2 Request parameters

Name

Type

Required

Description

clientId

String

Y

client\_id from [Create application](/CreateApplication)

accessToken

Y

date

Long

Y

Current time (timestamp in millisecond)
